# 更新日志

本文档记录 `coblog-frontend` 的主要变更。

---

## 2026-06-30

### 新增

- 新增暗色 / 亮色主题切换系统，并持久化用户主题偏好。
- 新增全局主题组合式工具 `src/composables/useTheme.js`。
- 新增搜索页筛选能力，支持关键词、分类、标签组合查询。
- 新增搜索页与首页的无限滚动加载体验。
- 新增写文章页面 `src/pages/write/`，支持 Markdown / HTML 双模式写作。
- 新增文章相关接口封装 `src/api/article.js`，覆盖发文、图片上传和 Markdown 预览。
- 新增 `/write` 路由，登录后可进入写作页。

### 调整

- 调整导航栏、页脚、首页、文章页、个人页、登录注册页等样式，统一适配主题变量。
- 调整文章详情页与时间线组件的交互，支持按分类跳转到文章列表筛选结果。
- 调整前端 Docker 构建流程，改为多阶段构建并使用 Nginx 承载静态资源。
- 更新 `README.md`，补充当前功能、环境变量与部署说明。

### 修复与优化

- 修复切换到文章列表模式时数据未重新加载的问题。
- 优化 RSS 跳转逻辑与路由处理。
- 优化移动端显示与部分页面样式细节。
- 统一登录 Token / RSS Token 的本地存储逻辑，改善登录态管理。

### 升级提示

- `VITE_API_BASE_URL` 应配置为后端根地址，例如 `http://localhost:8080`，不要额外拼接 `/api`。
- 若部署为单页应用，需要在 Nginx 中启用路由回退到 `index.html`。
